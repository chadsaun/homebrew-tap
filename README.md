# Homebrew Tap

This is a Homebrew tap for tools by @chadsaun.

## Installation

```bash
brew tap chadsaun/tap
brew install gchat
```

## Available Formulas

### gchat

Google Chat CLI - Send messages from the command line.

```bash
brew install chadsaun/tap/gchat
```

Or install directly via pip/pipx:

```bash
# Using pip
pip install gchat-cli

# Using pipx (recommended)
pipx install gchat-cli
```

See [gchat on GitHub](https://github.com/chadsaun/gchat) for documentation.
